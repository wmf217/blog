#### 柯里化的应用场景
假设有一个数组，要对其中每一个元素进行同样操作，比如加一或加二
```js
const arr1 = [0, 1, 2, 3];
let   arr2 = arr1.map(e => e+1); //加一
let   arr3 = arr1.map(e => e+2); //加二
```
那么是根据情况加不同的值，又不依赖全局变量，怎么处理，于是想把map里的1，2变成参数，但是通过map无法传递该参数，以下一个简单写法
```js
let   func = function (arr) {
    arr
}
const arr1 = [0, 1, 2, 3];
let   arr2 = arr1.map(e => e+1); //加一
```
